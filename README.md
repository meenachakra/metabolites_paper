# README

This repository contains code accompanying the Chakraborty et al. manuscript from Ami Bhatt’s lab, which focuses on the impact of age-related microbiome metabolites on the mammalian brain.

**Each folder in this repository corresponds to a different group of analyses from the manuscript and contains a separate `README_for_upload.pdf` describing the folder’s contents, including the inputs and outputs of the associated scripts. Please download PDF files to enable viewing.**
