# README

This repository contains code accompanying the Chakraborty et al. paper from Ami Bhatt’s lab, which focuses on the impact of age-related microbiome metabolites on the mammalian brain.

Each folder in this repository corresponds to a different group of analyses from the paper and contains a separate `README_for_upload.pdf` describing the folder’s contents, including the inputs and outputs of the associated scripts.

bioRxiv link to preprint: https://www.biorxiv.org/content/10.1101/2025.10.03.680371v2
